Library Management System
A simple command-line based Library Management System written in C++. This system allows users to add, search, update, and delete books from a database. It also provides a list of all available books. The system uses a binary file to store book information.

Features:
Add a book with name, author, and ISBN number
Search for a book by name, author, or ISBN number
Update a book's information
Delete a book from the database
List all available books

Usage:
Compile the program using g++ or another C++ compiler
Run the program using ./libsys.exe (or the name of the compiled executable)
Use the command-line interface to interact with the system
